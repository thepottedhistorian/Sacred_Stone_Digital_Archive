## 📜 STEWARDSHIP GUIDELINES
This archive is maintained by the Barony of the Sacred Stone Webministry. All code, scripts, and web assets hosted within this repository are intended for the service of the Society for Creative Anachronism (SCA).

### Archive Standards:
*   **Security:** Sensitive keys and identifiers are handled via environment variables/Script Properties and are not stored in plaintext.
*   **Documentation:** Every project directory contains its own `README.gs` or `README.md` manifest for technical clarity.
*   **Maintenance:** Logic is reviewed and updated periodically to ensure compliance with current Kingdom and Society digital policies.
